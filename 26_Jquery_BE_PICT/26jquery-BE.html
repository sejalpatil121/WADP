PICT B.E. Course Structure

First Year (FE)

Engineering Mathematics-I

Fundamentals of calculus and linear algebra.

Engineering Physics

Concepts of mechanics, optics, and waves.

Engineering Graphics

Technical drawing and orthographic projections.

Second Year (SE)

Data Structures

Introduction to arrays, linked lists, trees, and graphs.

Object-Oriented Programming

Concepts of OOP using C++ or Java.

Digital Electronics

Logic gates, flip-flops, and circuit design basics.

Third Year (TE)

Database Management Systems

Relational databases, SQL, and transactions.

Computer Networks

Network models, protocols, and security.

Software Engineering

Development lifecycle, design models, and testing.

Final Year (BE)

Machine Learning

Algorithms and models for predictive analytics.

Cloud Computing

Cloud architecture, services, and deployment.

Project Work

Capstone project integrating technical and soft skills.
